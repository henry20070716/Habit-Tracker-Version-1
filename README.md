## ✨ Features

- ✅ Add new habits
- ✅ Mark habits as completed
- ✅ Delete habits
- ✅ Track completed and pending habits
- ✅ Real-time progress bar
- ✅ Local Storage support
- ✅ Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API

## 🎯 How It Works

1. Enter a habit.
2. Click **Add**.
3. Mark habits as completed.
4. Delete habits when needed.
5. Track progress using the dashboard.
6. Data is saved automatically in Local Storage.
